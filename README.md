# worker-ayx-archive

本仓库用于归档和发布多个独立 HTML 页面。这些页面通常作为静态资源托管，不依赖特定域名或后端服务。

## 项目简介

`worker-ayx-archive` 是一个通用 HTML 页面归档仓库。仓库内所有页面均为独立、自包含的 HTML 文件，可直接通过浏览器或静态托管服务访问。本仓库不针对任何具体网站或域名，仅作为页面存档与分发用途。

## 目录结构

```
worker-ayx-archive/
├── README.md
├── index.html          (可选入口页)
├── pages/              (归档页面存放目录)
│   ├── page-001.html
│   ├── page-002.html
│   └── ...
└── assets/             (页面引用的静态资源，如 CSS、JS、图片)
    └── ...
```

- `pages/`：存放所有归档的 HTML 页面文件。
- `assets/`：存放页面所需的静态资源文件，按需组织。
